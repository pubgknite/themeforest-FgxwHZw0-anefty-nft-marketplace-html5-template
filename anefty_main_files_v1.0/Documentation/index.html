Anefty || NFT Marketplace HTML5 Template.

Version 1.0

Anefty is a NFT Marketplace HTML5 Template. It’s made with the latest Technology like SCSS + Gulp, Bootstrap and Jquery3. Anefty is the best HTML template for any kind of NFT Marketplace, digital marketplace, crypto-collectibles, digital art websites, creating friendly community with hundreds of members. well sorted documentation makes it very easy to customize.

It’s 100% super responsive and it works nicely on smartphones, tablets, & desktops. All files code have been well organized and nicely commented.

Installation

To begin installation please follow this guide bellow:

Via FTP:
  1. Open up your 'FTP Software' of choice and connect to your hosting
  2. Browse to required directory (Usually: ./public_html)
  3. Unzip the 'main_files.zip' and choose templates to upload.
  4. Upload the template to the server.

HTML Structure

The "Anefty" template is a responsive HTML5 template based on Bootstrap. The structure is based on a flexible grid that lets you change HTML pages quickly and logically with a nestable system.

If you would like to edit the layout of any elements in the whole template, you would do the following:

  • 1. Open any '.html' or '.css' file in a text editor.
  • 2. Find related comment/code.
  • 3. Do your magic.
